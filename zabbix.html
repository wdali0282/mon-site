zabbix - Dali Walid

uniquement pour ubuntu 24.04

Zabbix 7.0 TLS

se mettre en root

sudo -s

Installer le dépôt Zabbix

1: wget https://repo.zabbix.com/zabbix/7.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_latest_7.0+ubuntu24.04_all.deb

2: dpkg -i zabbix-release_latest_7.0+ubuntu24.04_all.deb

3: apt update

Installer le serveur Zabbix

apt install zabbix-server-mysql zabbix-frontend-php zabbix-apache-conf zabbix-sql-scripts zabbix-agent

Créer une base de données

1: mysql -uroot -p

2: mysql> create database zabbix character set utf8mb4 collate utf8mb4_bin;

3: mysql> create user zabbix@localhost identified by 'password';

4: mysql> grant all privileges on zabbix.* to zabbix@localhost;

5: mysql> set global log_bin_trust_function_creators = 1;

6: mysql> quit;

7: zcat /usr/share/zabbix-sql-scripts/mysql/server.sql.gz | mysql --default-character-set=utf8mb4 -uzabbix -p zabbix

8: mysql -uroot -p

9: mysql> set global log_bin_trust_function_creators = 0;

10: mysql> quit;

Configurez la base de données pour le serveur Zabbix

sudo nano /etc/zabbix/zabbix_server.conf

chercher:

#DBPassword=

et réécriver en dessous sans le #: DBPassword=votre_mdp

et pour quitter c'est: ctrl X

Démarrer les processus serveur et agent Zabbix

1: systemctl restart zabbix-server zabbix-agent apache2

2: systemctl enable zabbix-server zabbix-agent apache2

http://host/zabbix

dans le terminal taper hostname et metter le a la place de host dans l'url

Commencez à utiliser Zabbix
